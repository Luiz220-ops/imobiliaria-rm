Sistema de Orçamento de Aluguel Mensal - Imobiliária R.M

Este projeto foi desenvolvido como parte da avaliação da disciplina de **Algoritmos e Programação Orientada a Objetos** da UniFECAF. O objetivo da aplicação é automatizar e padronizar a geração de orçamentos mensais de locação para três categorias de imóveis: Apartamentos, Casas e Estúdios.

Funcionalidades:

- Cálculo Dinâmico de Aluguel: Processamento de valores com base no tipo de imóvel, quantidade de quartos e vagas de garagem.
- Regras de Negócio Aplicadas: Desconto de 5% para apartamentos caso o locatário não resida com crianças.
  - Cobrança incremental para vagas sobressalentes em estúdios.
- Parcelamento de Contrato: Amortização da taxa fixa de contrato institucional (R$ 2.000,00) em até 5 vezes.
- Exportação de Dados: Geração automática de um cronograma financeiro de 12 meses em formato `.csv`.

Tecnologias Utilizadas:

- Python 
- Paradigma de Orientação a Objetos (POO): Utilização de classes, herança e encapsulamento.
- Biblioteca CSV: Para persistência e exportação dos dados.

Gestão do Projeto (Metodologia Ágil)

O desenvolvimento deste software foi gerenciado utilizando a metodologia ágil **Kanban**, implementada diretamente através do **GitHub Projects**. As tarefas foram divididas nas colunas de *A Fazer*, *Em Progresso* e *Concluído* para garantir a rastreabilidade das entregas.
